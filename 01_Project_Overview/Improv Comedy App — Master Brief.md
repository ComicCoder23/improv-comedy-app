# Improv Comedy App — Master Brief

## Project Name
Improv Comedy App

## One-Line Description
A web-first community platform for discovering improv and comedy events, classes, teams, and performers — starting in Central Scotland.

## Purpose
To bring together the scattered improv and comedy ecosystem into one accessible, community-driven hub. Reduce friction of finding events, classes, teams, and collaborators. Lower the barrier for beginners. Give organisers and teachers better visibility without the noise of social media.

## Target Community
- **Phase 1:** Central Scotland (Glasgow, Edinburgh, Stirling, Kirkcaldy, Falkirk)
- **Phase 2:** UK-wide expansion after MVP validation

## Core Users

| User Type | Description |
|---|---|
| Beginner improvisers | People new to improv looking for entry points — jams, taster sessions, beginner classes |
| Experienced improvisers | Active performers seeking jams, festivals, collaboration, and growth opportunities |
| Teams / Troupes | Established groups needing public profiles, gig listings, and recruitment tools |
| Teachers | Workshop leaders and class instructors needing better visibility and booking routes |
| Venues / Studios | Spaces that host comedy/improv needing discoverability and listing management |
| Festival organisers | Running multi-day events with deadlines, submissions, and performer coordination |
| Audience members | People who want to watch improv/comedy shows and find tickets easily |

## Core Problem
Comedy and improv information is scattered across Instagram, Facebook, WhatsApp, Eventbrite, venue websites, personal networks, and word of mouth.

There is no single place to:
- Find what is on this week
- Discover who teaches locally
- See which venues host shows
- Find out how to get involved as a beginner
- Know when festival deadlines are
- Connect with collaborators

## Core Solution
A web-first community platform covering:
- Events listings (shows, jams, workshops, festivals)
- Class and workshop directory
- Team and troupe profiles
- Performer profiles
- Venue and studio listings
- Collaboration board
- Ticket and booking links
- Comedy discovery and search

## MVP Recommendation
Start with a **Central Scotland Improv Directory** before attempting a full social network.

Validate with:
- Manual data entry
- User-submitted events
- Community outreach to existing groups

Expand only when core value is proven and community trust is established.

## Build Principles
- **Low-cost:** Use free/freemium tools as long as viable
- **Permission-based:** No scraping without explicit consent
- **Web-first:** No app store dependency in Phase 1
- **Reusable:** Every asset built as a template or system
- **AI-assisted:** Use AI for content, categorisation, and admin
- **No aggressive scraping:** Store public links, not copied content
